- 书生·浦语大模型全链路开源开放体系介绍
- 轻松分钟玩转书生·浦语大模型趣味 Demo
- 基于 InternLM 和 Langchain 搭建个人知识库
- XTuner 大模型单卡低成本微调实战
- LMDeploy 大模型量化部署实践
- OpenCompass 大模型评测解读及实战
